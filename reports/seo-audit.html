04Ações priorizadas

Em ordem de impacto/esforço. Primeiros 4 são <30 min de trabalho, ganho SEO imediato.

Crítico
Criar robots.ts e sitemap.ts
Sem esses arquivos, o Google tem dificuldade em rastrear e indexar o site. É o fundamento da visibilidade orgânica.
Fix: criar src/app/robots.ts e src/app/sitemap.ts usando APIs nativas do Next 16 (MetadataRoute). Já existe esse padrão no makeone-site como referência.
Crítico
Gerar e referenciar og-image.png (1200×630)
Preview em WhatsApp, LinkedIn e redes — hoje compartilhar o link só mostra texto. Taxa de clique cai 40-60% sem preview visual.
Fix: criar imagem estática em public/og-image.png ou usar ImageResponse do next/og para geração dinâmica. Atualizar metadata.openGraph.images em layout.tsx.
Alto
Adicionar Structured Data (JSON-LD)
Schema ProfessionalService ou LocalBusiness habilita rich snippets no Google (estrelas, endereço, horários, área atendida) — diferencial vs concorrentes que não fazem.
Fix: injetar JSON-LD no layout.tsx com dados da M2B (nome, fundação 1996, área de atuação "Brasil", disciplinas como hasOfferCatalog, depoimentos como review).
Alto
Canonical URL
Quando o domínio próprio for apontado (ex: m2bprojetos.com.br), conteúdo duplicado entre vercel.app e domínio final pode canibalizar ranking.
Fix: adicionar alternates: { canonical: 'https://m2bprojetos.com.br' } no metadata do layout.tsx.
Médio
Favicon + Apple Touch Icon
Branding em abas do navegador, favoritos e ícone na home de iOS/Android. Hoje mostra o favicon default do Next.js.
Fix: gerar public/favicon.ico, public/icon.png (512×512) e public/apple-icon.png (180×180) a partir do monograma M2B. Next 16 auto-gera as meta tags.
Médio
Habilitar Vercel Speed Insights
Coleta de Core Web Vitals com usuários reais (field data) — melhor que testes sintéticos. Mostra no dashboard Vercel e é usado pelo Google para ranking (Chrome UX Report).
Fix: pnpm add @vercel/speed-insights + <SpeedInsights /> no root layout. Plano Pro da Vercel inclui gratuitamente.
Médio
Content & keywords: SEO de entrada
O site hoje não tem páginas de disciplina individuais. Quem busca "projeto hidráulica predial BIM" não acha M2B porque não existe página dedicada com esse termo em destaque.
Fix (fase 2): criar rotas /disciplinas/hidraulica, /disciplinas/ppci, etc — cada uma com ~800 palavras de conteúdo técnico. Gera long-tail SEO e páginas de aterrissagem específicas pra campanhas.
